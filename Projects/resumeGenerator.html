Resume Generator

Resume generator that generates a LaTeX resume based on the job description by looking at your skills file.

Python LaTeX Gemini API Streamlit

Overview

I've found it pretty time consuming to keep on editing each resume by hand for each job application. I didn't want to pay someone for their service so I built my own.

The part that I find most useful is the skills file. You can list all of your projects and skills and it will choose only the applicable skills for the job posting. This way you don't have to remember what you did for each job posting.

Key Features

  • Generates a LaTeX resume based on the job description
  • Chooses only the applicable skills for the job posting
  • Uses Gemini API to generate the resume
